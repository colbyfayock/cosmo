# Weather Skill

*Get weather via @cosmo/lib (Open-Meteo)*

## Usage

Import from lib in scripts:
```typescript
import { getWeather } from '@cosmo/lib';
const weather = await getWeather(40.02, -75.64);
```

## Package

`packages/lib/weather.ts`

## Default Location

Exton, PA (19341): lat 40.02, lon -75.64
