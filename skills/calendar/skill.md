# Calendar Skill

*Access Google Calendar via gog CLI*

## Usage

```bash
# Get help
gog -h
gog calendar -h
```

## Examples

```bash
# List events
gog calendar events
gog calendar events --calendar "work"

# Create event
gog calendar event create --title "Meeting" --start "2026-02-22T14:00:00"
```
