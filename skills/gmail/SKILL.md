# Gmail Skill

*Access Gmail via gog CLI.*

## Usage

See `gog` skill for full documentation.

```bash
# Get help
gog gmail -h
```

## Examples

```bash
# List emails
gog gmail list

# Send email
gog gmail send --to "email@example.com" --subject "Subject" --body "Body"

# Search
gog gmail search "query"
```
