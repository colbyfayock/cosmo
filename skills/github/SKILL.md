# GitHub Skill

*GitHub CLI for repos, issues, PRs, and more.*

## Usage

```bash
# Get help
gh --help

# Command-specific help
gh repo --help
gh pr --help
```

## Quick Examples

```bash
# Auth status
gh auth status

# List repos
gh repo list

# Create PR
gh pr create --title "Fix bug" --body "Description"
```

## Docs

See `gh --help` or visit https://cli.github.com/
